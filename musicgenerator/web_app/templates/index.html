What We Do

MusicGen is a program that generates music on the fly. By taking in a tempo from the user, it uses randomized music generation algorithms to output music for the user.

The user inputs a tempo using MusicGen's mobile integration; by tapping on their phone to a beat the user can set a tempo for their music. The algorithm then adjusts the music to the user defined tempo.

The innate randomness promises that no two tracks will be the same and that the tune will be perfectly calibrated to the user's needs.
